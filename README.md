# CSYE6225-Cloud Computing
Web application built with Spring Boot

## Build Instructions
-  Clone this repository  into the local system
-  Open the CLI
-  mvn clean install
-  java -jar jar file

## Run the test
-  mvn test

## Kill the process on a port
-  kill $(lsof -t -i:8080)
